James Socol

James is an engineer and leader with experience on all parts of the web application stack. He is an engineering manager at Condé Nast Co/Lab, and previously ran TodaysMeet, the premier backchannel chat platform for educators.

Previously, James was a platform engineer at The Groundwork, an API evangelist and application developer at Bitly and a web engineering manager and web security liaison at Mozilla. He is an active member of the Python and JavaScript communities and maintains several popular open source software projects.

James blogs at Coffee on the Keyboard.
